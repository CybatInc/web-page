User Guide_

The User Guide provides an in-depth reference to the Cybat Engine logic. Learn how to tune AI parameters, build custom threat profiles, and analyze the telemetry dashboard.

1. Navigating the Telemetry Dashboard

  • Global View: See a high-level overview of incoming logs, active alerts, and autonomous remediations taken by the engine.
  • Threat Analytics: Deep-dive into specific anomalies identified by the Gemini models, with detailed contextual metadata surrounding the event.

2. Configuring Environments

  • Project Mapping: Manage the GCP projects you have connected via Pub/Sub sinks. Organize them into Folders or specific monitored zones.
  • Alerting Integrations: Connect your Cybat Workspace to external alerting tools like Slack, PagerDuty, or native Google Cloud Monitoring components.

3. Generating Threat Reports

  • Automated Summaries: Configure the engine to generate weekly security baseline reports to share with compliance or administrative stakeholders.
  • On-Demand Export: Export specific incident timelines and AI analysis logs into standard formats for post-mortem tracking.
